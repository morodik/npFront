Passwordless аутентификация на основе Ed25519 + JWT.
Вместо пароля криптографическая подпись. Приватный ключ никогда не покидает устройство пользователя.
Репозитории
Backend: https://github.com/morodik/npAuth
Frontend: этот репозиторий

Стек
Backend: 
Go — net/http
PostgreSQL — хранилище пользователей и челленджей
GORM — ORM и автомиграции
Ed25519 — криптографические ключи
JWT — сессионные токены

Frontend
React + Vite
@noble/ed25519 — генерация ключей и подпись в браузере
localStorage(ознакомительный вариант) - хранение приватного ключа на устройстве

Регистрация

Клиент генерирует пару ключей Ed25519
Публичный ключ отправляется на сервер
Приватный ключ сохраняется на клиенте в localStorage(в идеале на устройстве)  сервер его никогда не видит

Вход
Клиент запрашивает одноразовый challenge у сервера
Клиент подписывает challenge своим приватным ключом
Сервер проверяет подпись через публичный ключ из БД
При успехе - сервер пускает пользователя и выдаёт JWT токен

Переменные окружения
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=yourpassword
DB_NAME=authdb
JWT_SECRET=длинная_случайная_строка_минимум_32_символа
